# BYKS Satın Alma - Yayına Alma

Bu demo PWA olarak hazırdır. Vercel'e yüklendiğinde iPhone ve Android telefondan normal uygulama gibi ana ekrana eklenebilir.

## En kolay yol: Vercel

1. https://vercel.com adresinden giriş yapın.
2. `Add New Project` seçin.
3. Bu klasörü GitHub'a yüklediyseniz projeyi GitHub'dan seçin.
4. Framework otomatik olarak `Next.js` görünür.
5. Build komutu: `pnpm build`
6. Install komutu: `pnpm install`
7. `Deploy` butonuna basın.

Yayın bittikten sonra Vercel size buna benzer bir adres verir:

`https://byks-satin-alma.vercel.app`

## Telefona uygulama gibi ekleme

### iPhone

1. Safari ile Vercel adresini açın.
2. Paylaş butonuna basın.
3. `Ana Ekrana Ekle` seçin.
4. Uygulama adı `BYKS Satın Alma` olarak görünecektir.

### Android

1. Chrome ile Vercel adresini açın.
2. Sağ üst menüden `Ana ekrana ekle` veya `Uygulamayı yükle` seçin.
3. Uygulama ikonla açılır.
